# Portfólio — Rodrigo Viana Quirino

Site portfólio profissional em Next.js, estilo currículo interativo.

## Desenvolvimento

```bash
npm install
npm run dev
```

Abra [http://localhost:3000](http://localhost:3000).

## Build

```bash
npm run build
npm start
```

## Conteúdo

Os dados do perfil ficam em `src/data/profile.ts`. O PDF do CV está em `public/cv-rodrigo-viana-quirino.pdf`.

## Deploy

Recomendado: [Vercel](https://vercel.com) — conecte o repositório e faça deploy automático.
